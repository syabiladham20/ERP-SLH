Help & Usage Guide

  • Create Flock: Go to 'Manage Flocks'. Select an existing house or type a new name (e.g., VC1) to create one. Enter intake details.
  • Phase: Use the 'Start Prod' button on the Dashboard to switch a flock from Rearing to Production.
  • Close Flock: Use the 'Close' button to end a flock cycle. Closed flocks can be viewed in the 'Manage Flocks' history list.

  • Feed: Enter the amount given in Grams per Bird.
  • Water: Enter the 3 meter readings as whole numbers (e.g., 10500 for 105.00). The system calculates consumption based on the difference from the previous day.
  • Edit: You can correct any entry by going to the Flock History page and clicking 'Edit' next to the log.

You can import historical data from Excel files (.xlsx). The file can contain multiple sheets (one for each flock).

Requirements:

  • Header must contain HOUSE, INTAKE DATE, MALE/FEMALE INTAKE in the first 5 rows (Column B).
  • Daily data table must start at Row 10.
